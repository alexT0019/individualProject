Why did you started learning web development?

The first time I met coding was in Secondary School. I learnt to use Visual Basic to create a game for my project in ICT classes, which I found it interesting and satisfying.

I also learnt to design websites and apps in University, which were great experiences for me as part of web development.

After graduating from University, I started working as a clerk, but then I realised I need something more challenging and creative, which I started learning coding and enrolled for the web development course organised by Cristian Action.

My Goal

My goal is to create user-friendly website and applications to help the others to solve problems they encounter in their daily life. It is meaningful for me to contribute to the society. Not just about helping people, but also to fulfill the need of self-actualization.
